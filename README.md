# 长沙理工大学校园网登录脚本

一键or自动化登录校园网，从此省去等待浏览器和输入账号密码的烦恼。

**使用前**:

> 请在**config.ini**里按照提示进行配置
>
> 请安装必要的第三库:
>
> `pip install -r requirement.txt`

**Windows**:

> 双击 **login.cmd** 即可

**Linux**:

> 帮助: `python main.py help`
>
> 登录: `python main.py login`
>
> 注销: `python main.py logout`

想要实现开机自启，请自行goole。
